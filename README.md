# 🧮 Simple Calculator

A simple calculator built using Python.

## ✨ Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Percentage (%)
- Modulus (%)
- Power (x^y)
- Square (x²)
- Cube (x³)
- Input Validation
- Error Handling using try-except

## 📚 Concepts Used

- Variables
- Input & Output
- if-elif-else
- while Loop
- try-except
- float() & int()
- Arithmetic Operators

## ▶️ How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run:

```bash
python calculator.py
```
